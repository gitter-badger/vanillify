Luxaura.js comes with syntax highlighting for Markdown files. Here are some examples:

HTML
````html
<!DOCTYPE html>
<html>
<head>
<title>Hello, world!</title>
</head>
<body>
<h1>Hello!</h1>
</body>
</html>
````
Markdown
````markdown
# hello, there!
**bold**
*italic*
## h2
````
YAML
````yaml
title: your awesome title
````
JavaScript
````javascript
var abc = "xyz"
console.log(abc);
````
